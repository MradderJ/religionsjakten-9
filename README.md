# Religionsjakten 9

En fristående gamifierad träningsapp för religionskunskap i årskurs 9.

Appen är byggd som en enda statisk HTML-fil och kör Pythonlogik i webbläsaren via PyScript/Pyodide. Den kan hostas på GitHub Pages och öppnas från mobil, surfplatta eller dator.

## Innehåll

- Frågor om världsreligioner, etik, livsfrågor, sekularisering, religionsfrihet och källkritik.
- XP, nivåer, streaks, märken och dagliga missions.
- Mini-essä med checklista för att träna utvecklade resonemang.
- Ljust och mörkt tema.
- Mobilanpassad layout.

## Viktigt

Detta är inte officiellt material från Skolverket och ska ses som ett fristående träningsstöd.

Appen samlar inte in personuppgifter och kräver ingen inloggning. Den laddar PyScript/Pyodide och typsnitt från externa CDN:er, så internetuppkoppling krävs.

## Körning

Öppna `index.html` i en webbläsare, eller publicera repot med GitHub Pages.
